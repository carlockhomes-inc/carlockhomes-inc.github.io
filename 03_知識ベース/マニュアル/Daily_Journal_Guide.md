# 日誌システム（Daily Journal）の使い方

「第2の脳」の中核となる日誌システムの使い方です。
AIエージェントと連携して、日々の活動を記録・分析・資産化するためのサイクルを回します。

## 1. 1日の流れ

### 🌞 朝：1日を始める (`/today-start`)
1日の始まりに、チャットで `/today-start` と入力してください。
システムが以下のことを自動で行います：
1.  **新しい日誌の作成**: 今日の日付のファイルを作成します。
2.  **タスクの引き継ぎ**: 昨日の未完了タスクを自動で今日に移動します。
3.  **秘書のブリーフィング**: コンテキストに基づき、今日意識すべきことを提案します。
4.  **戦略立案 (Big 3)**: 今日の最重要タスク（Big 3）を定義します。

### 🏃 日中：記録する
日誌ファイルを開き、自由に書き込んでください。

*   **💡 思いつきメモ / Inbox**: ふと思いついたアイデア、感情、タスクなどを箇条書きでメモします。整理は後でAIがやるので、殴り書きでOKです。
*   **✅ Today's Tasks**: 今日のタスクの進捗を管理します（完了したら `[x]` にする）。

### 🌙 夜：1日を終える (`/today-finish`)
1日の終わりに、日誌の下部にある **「🧠 思考と感情のログ (Context Stream)」** に、今日あったことや感じたことを音声入力などで一気に書き出してください。
その後、チャットで `/today-finish` と入力します。
システムが以下のことを行います：
1.  **ログの分析**: 書き出したログを整理・構造化します。
2.  **資産の抽出**: ログから「アイデア」「価値観」「スキル」「実績」を抽出し、記録します。
3.  **コンテキスト更新の提案**: あなたの変化に合わせて、プロフィールの更新案を提示します。

---

## 2. 日誌の構成（セクション解説）

| セクション | 説明 |
| :--- | :--- |
| **💡 思いつきメモ / Inbox** | 一時的なメモ置き場。未分類のタスクやアイデアを放り込みます。 |
| **🌞 今日が始まるよ！** | 朝のセットアップ用エリア。コマンドメモなどが書かれています。 |
| **☕ 秘書のブリーフィング** | AIからの朝のメッセージ、リマインダー、提案が表示されます。 |
| **✅ Today's Tasks** | 今日やるべきタスクリスト。 |
| **📥 未分類 (Inbox / Task Pool)** | 昨日から引き継がれたタスクなどが一時的に置かれます。 |
| **🎯 翌日以降...** | 今日はやらないけれど、近いうちにやるタスク。 |
| **🌙 今日を終えるよ！** | 夜の振り返り用エリア。 |
| **🧠 思考と感情のログ** | **【重要】** 脳内のデトックス場所。AIに教えたいことを全てここに吐き出します。 |
| **💎 抽出された情報** | `/today-finish` 後に、AIが抽出した「宝物（資産）」がここに蓄積されます。 |

## 3. 活用のコツ
*   **AIに話しかけるように書く**: 「🧠 思考と感情のログ」は、誰かに見せる文章ではなく、AI（もう一人の自分）への共有として本音で書いてください。
*   **コマンドを忘れない**: 手動でファイルをコピーしたりする必要はありません。すべてコマンド（`/today-start`, `/today-finish`）に任せてください。
