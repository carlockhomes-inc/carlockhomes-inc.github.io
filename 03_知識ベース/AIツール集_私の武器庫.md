# 🛠️ 文子さんのAI武器庫（ツール集）

ビジネスや生活を助けてくれる、頼れるAIパートナーたちのリストです。
新しいツールを見つけたら、ここに追記していきましょう。

## 🔍 お仕事・検索・リサーチ

### 1. Genspark（ジェンスパーク）
*   **URL**: [https://www.genspark.ai/](https://www.genspark.ai/)
*   **得意なこと**: 
    *   調べ物をして、まとめてページにしてくれる（資料作成の下書きに最適）
    *   LP（ホームページ）の構成案や文章を一瞬で考える
    *   「〇〇について教えて」と聞くと、複数のサイトを読んで要約してくれる
*   **文子さんの使い方**: 
    *   「おまつさんのスクール」のLP構成案を作成
    *   「カーロックホームズ」の企業サイト構成を作成

### 2. Perplexity（パープレキシティ）
*   **URL**: [https://www.perplexity.ai/](https://www.perplexity.ai/)
*   **得意なこと**: 
    *   Gensparkに似ているが、もっと「検索」に特化している
    *   情報のソース（元ネタ）をしっかり示してくれる

## 📄 資料作成・スライド

### 3. Gamma（ガンマ）
*   **URL**: [https://gamma.app/](https://gamma.app/)
*   **得意なこと**: 
    *   「タイトル」を入れるだけで、美しいパワーポイント（スライド）資料を作る
    *   Webページのような資料も作れる
*   **文子さんの使い方**: 
    *   会社でのプレゼン資料、説明資料の作成に

### 4. Napkin（ナプキン）
*   **URL**: [https://napkin.ai/](https://napkin.ai/)
*   **得意なこと**: 
    *   文章をコピペすると、一瞬で「図解（チャート）」にしてくれる
    *   文字ばかりの資料にメリハリをつけたい時に最強

## 🎨 デザイン・画像

### 5. Canva（キャンバ）
*   **URL**: [https://www.canva.com/](https://www.canva.com/)
*   **得意なこと**: 
    *   チラシ、名刺、SNS画像の作成
    *   最近はAI機能も充実している

## 🧠 パートナー・思考の整理

### 6. Antigravity（私 / Second Brain）
*   **場所**: このPCの中（VS Code / Obsidian）
*   **得意なこと**: 
    *   文子さんの価値観・文脈を深く理解している
    *   メルマガの執筆、日誌の記録、話し相手
    *   HTML/CSSのコーディング（Gensparkが作った設計図を組み立てる）
    *   **「私だけは、文子さんだけの秘密のパートナー」**

---

## 📝 使い方のアドバイス

**「二刀流」がおすすめです！**

1.  **Genspark**などのWebツールで、パパッと「材料（構成案やコード）」を作る。
2.  その材料を**Antigravity（私）**に渡して、「仕上げ」や「保存」をする。

この流れなら、文子さんの「思考の深さ」と、Webツールの「手軽さ」の両方を活かせます！
