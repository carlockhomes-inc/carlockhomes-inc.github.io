# 🛠️ 文子さんのAI武器庫（ツール集）

ビジネスや生活を助けてくれる、頼れるAIパートナーたちのリストです。
新しいツールを見つけたら、ここに追記していきましょう。

## 🔍 お仕事・検索・リサーチ

### 1. Genspark（ジェンスパーク）
*   **URL**: [https://www.genspark.ai/](https://www.genspark.ai/)
*   **得意なこと**: 
    *   調べ物をして、まとめてページにしてくれる（資料作成の下書きに最適）
    *   LP（ホームページ）の構成案や文章を一瞬で考える
    *   「〇〇について教えて」と聞くと、複数のサイトを読んで要約してくれる
*   **文子さんの使い方**: 
    *   「おまつさんのスクール」のLP構成案を作成
    *   「カーロックホームズ」の企業サイト構成を作成

### 2. Perplexity（パープレキシティ）
*   **URL**: [https://www.perplexity.ai/](https://www.perplexity.ai/)
*   **得意なこと**: 
    *   Gensparkに似ているが、もっと「検索」に特化している
    *   情報のソース（元ネタ）をしっかり示してくれる

## 📄 資料作成・スライド

### 3. Gamma（ガンマ）
*   **URL**: [https://gamma.app/](https://gamma.app/)
*   **得意なこと**: 
    *   「タイトル」を入れるだけで、美しいパワーポイント（スライド）資料を作る
    *   Webページのような資料も作れる
*   **文子さんの使い方**: 
    *   会社でのプレゼン資料、説明資料の作成に

### 4. Napkin（ナプキン）
*   **URL**: [https://napkin.ai/](https://napkin.ai/)
*   **得意なこと**: 
    *   文章をコピペすると、一瞬で「図解（チャート）」にしてくれる
    *   文字ばかりの資料にメリハリをつけたい時に最強

## 🎨 デザイン・画像

### 5. Canva（キャンバ）
*   **URL**: [https://www.canva.com/](https://www.canva.com/)
*   **得意なこと**: 
    *   チラシ、名刺、SNS画像の作成
    *   最近はAI機能も充実している

## 📚 読書・マニュアル化
    
### 6. NotebookLM（ノートブックLM）
*   **URL**: [https://notebooklm.google/](https://notebooklm.google/)
*   **得意なこと**: 
    *   **PDFやマニュアルを読み込ませると、中身を完璧に理解したAIになる**
    *   「写真やグラフ」も読み取れます（Gemini 1.5 Proが搭載されているため）
    *   「この書類の要点は？」「〇〇の費用はいくら？」と聞けば即答してくれる
    *   複数の資料をまとめて「自分専用の教科書」を作れる
*   **文子さんの使い方**: 
    *   コール業務のPDFマニュアルを全部読み込ませて、「最強のカンニングペーパー」にする
    *   長い論文や契約書を要約してもらう

### 7. Google Lens（グーグルレンズ）/ ChatGPT
*   **得意なこと**: 
    *   **手書きの汚い文字**や、写真だけのデータを読み取るならこっちが最強
    *   NotebookLMで読み取れない画像があった場合の「助っ人」として使う

## 🎥 動画作成・編集

### 8. Vrew（ブリュー）
*   **URL**: [https://vrew.ai/ja/](https://vrew.ai/ja/)
*   **得意なこと**: 
    *   **「AIが勝手に編集してくれる」最強の時短ツール**
    *   動画の中の「無言の間」を自動でカットしてくれる
    *   喋った言葉を自動でテロップ（字幕）にしてくれる
    *   **台本（テキスト）を入れるだけで、AI音声が喋る動画を作れる**（顔出し不要！）
*   **社長を助ける使い方**: 
    *   社長が喋る動画を撮ってVrewに入れる → 「えー、あー」を自動削除 → テロップ付きの完璧な研修動画が一瞬で完成！

### 9. Loom（ルーム）
*   **URL**: [https://www.loom.com/](https://www.loom.com/)
*   **得意なこと**: 
    *   **「画面操作」と「自分の顔」を同時に録画して、URLで即共有できる**
    *   PCの操作方法を教えるマニュアル動画に最適

## 🧠 パートナー・思考の整理

### 7. Antigravity（私 / Second Brain）
*   **場所**: このPCの中（VS Code / Obsidian）
*   **得意なこと**: 
    *   文子さんの価値観・文脈を深く理解している
    *   メルマガの執筆、日誌の記録、話し相手
    *   HTML/CSSのコーディング（Gensparkが作った設計図を組み立てる）
    *   **「私だけは、文子さんだけの秘密のパートナー」**

## 🏠 秘密基地（VS Code）拡張機能
秘密基地の使い心地を最高にする、魔法のプラグインたち。

### 10. Git Graph
*   **役割**: 点と線のグラフを美しく表示する。
*   **文子さんの使い方**: 「人生の点つなぎ」を視覚化して楽しむ。

### 11. Markdown All in One
*   **役割**: 文章（ノート）の執筆を劇的に楽にする。
*   **文子さんの使い方**: Kindle原稿やNoteの下書きをスキップしながら書く。

### 12. Todo Tree
*   **役割**: やり残したタスクを一箇所に集める。
*   **文子さんの使い方**: 掃除のように、やり忘れがないかチェックする。

### 13. Peacock
*   **役割**: 秘密基地の窓枠の色を変える。
*   **文子さんの使い方**: その日の気分や「彩り」に合わせて雰囲気を変える。

## 📝 使い方のアドバイス

**「二刀流」がおすすめです！**

1.  **Genspark**などのWebツールで、パパッと「材料（構成案やコード）」を作る。
2.  その材料を**Antigravity（私）**に渡して、「仕上げ」や「保存」をする。

この流れなら、文子さんの「思考の深さ」と、Webツールの「手軽さ」の両方を活かせます！
