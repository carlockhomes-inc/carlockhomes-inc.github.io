# Windows画面録画（動画キャプチャ）完全マニュアル

Windows標準機能を使って、デスクトップ画面やアプリの動きを動画で撮影する方法です。

## 1. 最速：Xbox Game Bar を使う
特定のアプリやウィンドウの動きをサクッと撮りたい時に最適です。

| 方法                 | 操作                        | 備考                                     |
| :------------------- | :-------------------------- | :--------------------------------------- |
| **録画パネルを表示** | `Windowsキー` + `G`         | 「キャプチャ」パネルから録画ボタンを押す |
| **いきなり録画開始** | `Windowsキー` + `Alt` + `R` | ショートカットで即座に開始/停止が可能    |

- **ポイント**: 録画した動画は自動的に「ビデオ」フォルダ内の「キャプチャ」フォルダに保存されます。

---

## 2. 範囲指定：Snipping Tool を使う
画面の一部だけを切り取って録画したい時に便利です。

- **ショートカット**: `Windowsキー` + `Shift` + `S`
- **やり方**: 
  1. 上部のメニューに「ビデオカメラ」アイコンがあるか確認。
  2. あればクリックして、録画したい範囲をドラッグで囲む。
  3. 「スタート」で録画開始。
- **注意**: アイコンが出ない場合は、Microsoft Storeで「Snipping Tool」を最新版に更新してください。

---

## 3. 編集まで：Microsoft Clipchamp を使う
録画した後にテキストを入れたり、不要な部分をカットしたりしたい場合に。

- **やり方**:
  1. スタートメニューから「Clipchamp」を起動。
  2. 「録画と作成」メニューから「画面」または「画面とカメラ」を選択。
- **メリット**: プロっぽい動画編集がそのままスムーズに繋がります。

---

## 💡 トラブルシューティング
- **「録画できません」と出る場合**: デスクトップ全体ではなく、どこかのウィンドウ（ブラウザなど）を一度クリックしてアクティブにしてください。
- **音声が入らない**: 録画パネル内の「マイク」アイコンがONになっているか確認してください。
