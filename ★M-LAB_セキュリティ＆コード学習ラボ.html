🔍 1行目からの組み立て例：『魔法のあいさつボタン』

「魔法のあいさつボタン」を作るための3つのステップを、上から順番に見てみましょう。

【工程 1】 HTML で「姿」を置く

まずは、「ボタン」という部品を画面に置きます。

<!-- 部品を置いて、背番号「myBtn」をつける -->
<button id="myBtn">魔法をかける</button>

【工程 2】 CSS で「お化粧」をする

黄金色にして、M-LABらしく仕上げます。

/* 背番号「myBtn」の色を金色にする */
#myBtn {
  background: gold;
  padding: 10px 20px;
}

【工程 3】 JavaScript で「命」を吹き込む

最後に、「押したらメッセージを出す」というスイッチを入れます。

// myBtn を探して、クリックした時の動きを決める
document.getElementById('myBtn').onclick = () => {
  alert("M-LABへようこそ！");
};

この3つの積み重ねが、あなたの「2nd Brain（第2の脳）」を作ります！

▼ 【実物】実際に並べて書くと、こうなります！

（このボタンは、上の「3つの工程」だけでできています）

🎬 合体バージョン：『魔法のあいさつボタン』全コード

これまでに学んだ「土台・お化粧・命」の3つを、一つのファイルに書くとこうなります。
この順番で書くのが、アプリを動かすための「正しいお作法」です。

<!-- (1) 土台：HTMLで部品を置く -->
<button id="myBtn">魔法の挨拶</button>

<!-- (2) お化粧：CSSで見た目を整える -->
<style>
  #myBtn {
    background: gold;
    color: black;
    padding: 10px 20px;
    border-radius: 10px;
  }
</style>

<!-- (3) 命：JSで動きを入れる -->
<script>
  document.getElementById("myBtn").onclick = () => {
    alert("M-LABへようこそ！魔法の成功です！");
  };
</script>
