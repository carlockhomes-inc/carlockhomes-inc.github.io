# 🦁 社長へ：AI導入のご提案メモ

## 1. そもそも何を導入するの？
今回提案したいのは、**「Antigravity（アンチグラビティ）」** というAIツールです。

「うちにはもうGemini（Google Workspace）があるじゃないか？」
と思われるかもしれませんが、実は **役割が違います！**

## 2. Gemini と Antigravity の違い

| 特徴           | 🤖 Gemini (すでにある)                        | 🦁 Antigravity (入れたい)                                                                       |
| :------------- | :------------------------------------------- | :--------------------------------------------------------------------------------------------- |
| **役割**       | **「相談役」**                               | **「実務担当の部下」**                                                                         |
| **場所**       | インターネット（ブラウザ）                   | **パソコンの中 (VS Code)**                                                                     |
| **できること** | ・メールの下書き<br>・アイデア相談<br>・検索 | ・**マニュアルをファイルとして作る**<br>・**フォルダを整理する**<br>・**コードを書いて動かす** |

👉 **「Gemini」に相談して決めた内容を、「Antigravity」が手を動かして形にする。**
この2人が揃うと、最強のチームになります！

## 3. 必要な準備
導入には、会社のパソコンに以下の2つをインストールする必要があります。
（無料から始められます！）

1.  **VS Code (Visual Studio Code)**
    *   世界中のエンジニアが使っている「作業場」ソフトですが、これが私たち（AIと私）の仕事場になります。
2.  **Antigravity (拡張機能)**
    *   VS Codeの中に、AI（私）を住まわせます。

## 4. 期待できる効果
*   **事務作業の時間を短縮**（マニュアル作成など）
*   **私（松浦）が「現場」や「社長のサポート」に使える時間が増える！**

## 5. AI活用の先にある「未来の働き方」と「福利厚生」

AI（Antigravity）を入れて効率化するだけが目的ではありません。
そこで浮いたコストと時間を、**「社員とその家族の幸せ（Well-being）」** に還元する新しい会社作りを提案したいです。

### 💡 具体案：福利厚生サービス「ResortWorx（リゾートワークス）」の導入
[ResortWorx 公式サイト](https://pr.resortworx.jp/lp08/)

単なる保養所契約ではありません。これは**「第3の賃上げ」**とも呼ばれる、税金のかからない給与のようなメリットです。

*   **家族も使える！**: 社員だけでなく、そのご家族（二親等以内）も、全国の一流ホテルやリゾートを最大80%OFFで利用できます。
*   **「お父さんの会社、すごいね！」**: ご家族にそう言ってもらえることで、社員の会社への信頼と定着率（エンゲージメント）が圧倒的に高まります。
*   **採用の武器に**: 「40代からの採用」を掲げる私たちにとって、家族を大切にする制度は最強の採用ブランディングになります。

## 6. 採用戦略：なぜ今「社員」なのか？（外注との比較）

採用を強化するにあたり、「高い外注費を払い続ける」のと「社員を育てる」のでは、どちらが会社の未来にとってプラスか比較しました。

| 項目         | 🏢 外注（業務委託）                 | 🏠 正社員（今回の提案）                                 |
| :----------- | :--------------------------------- | :----------------------------------------------------- |
| **コスト**   | 短期的には安いが、繁忙期は高騰する | 固定給＋福利厚生だが、**AI活用で生産性を最大化**できる |
| **ノウハウ** | 人に依存し、会社に残らない         | **会社に蓄積される（資産になる）**                     |
| **帰属意識** | 希薄（掛け持ちが多い）             | **高い（家族も含めたファン化）**                       |
| **緊急対応** | 相手の都合次第                     | **優先的に対応可能**                                   |
| **福利厚生** | 対象外                             | **ResortWorxなどで家族ごと守れる**                     |

👉 **結論**：
「ResortWorx」等の福利厚生を充実させることで、**「外注にはない魅力（家族への安心）」**を提示でき、優秀で責任感のある40代人材を**社員として確保**できます。これが長期的な会社の安定成長につながります。

## 7. 教育革命：動画マニュアルで「見て覚える」研修へ

「未経験の40代を採用しても、教える時間が惜しい…」
その悩みも、Antigravity（と私）で解決します。

*   **動画マニュアルの作成**:
    *   ベテラン社員の「鍵交換作業」や「珍しい鍵の解錠」を撮影し、**社内用YouTube（研修動画）**として蓄積します。
    *   言葉だけでは伝わりにくい「手元の感覚」や「音」も、動画なら何度も見て復習できます。
*   **教育コストの削減**:
    *   「同じことを何度も教える時間」がゼロになります。新人さんはまず動画を見て予習し、現場で実践するサイクルを作れます。

**「技術を動画で可視化し、誰でも一流になれる仕組みを作る」**
これが最強の技術継承です。
