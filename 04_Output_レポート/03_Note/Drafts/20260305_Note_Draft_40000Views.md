# 誰だかわからない人の投稿を、よくぞ読んでくれました。もうすぐ4万ビューになる2020年からのNote

気がつけば、2020年から書き始めたこのNoteの全体ビュー数が、もうすぐ「4万回」に届こうとしています。

読んでくださった皆さん、
そして、温かい「スキ」をポチッと押してくださった3,991人の皆さん。
本当に、本当にありがとうございます！

今日、ふと自分の過去の記事をダッシュボードで見返してみました。
「NYから素敵な贈り物」「あなたらしく！を応援するライター純子さんへ」……。

当時、私はコンビニの店長をしていました。
一緒に働くスタッフたちのこと、日々お店で起こる様々な出来事から感じたこと。
文章の書き方も型も知らなくて、今みたいにAIの力なんて借りられなくて、ただただ、その時の熱量と想いだけで必死にタイピングして「大事な物語」を残そうとしていた記憶が蘇ります。

でも、今改めて一番強く思うのは、
**「こんな、当時は誰だかもわからない、ただの一人のコンビニ店長の書いた文章を、よくぞ読んで、スキまで押してくれたな…」**
という、驚きと感謝でいっぱいの気持ちです。

私は、あの不器用な記事たちを「恥ずかしいから」と消したりはしません。

なぜなら、その文章の中にこそ、今の私に繋がる「原点」が詰まっているからです。
体温があり、泥臭くもがき、人と繋がろうとしていた、あの時の私がそこにいるからです。

最近、私の「新しい目標」にまた一歩近づいた話もあったりして、新しい扉が次々と開こうとしています。

そこで私は、これから一緒に何かに挑戦しようとしている人たちに向けて、今の等身大の言葉でこう伝えようと決めました。

「最初は上手く書けなくていいんです。不器用でも、今のあなたの想いを残すことから始めましょう。それがいつか、私を見つけてくれたように、誰かの心を動かす財産になります」と。

6年前、たった一つの記事から始まったこの場所が、今では私に「分析する楽しさ」や「教える喜び」まで教えてくれています。

これからも、不器用な過去の自分も、AIと一緒に新しい波に乗る今の自分も、全部ひっくるめて「M-Lab」という場所から発信し続けていきます。

いつも応援ありがとうございます！
そして、これからもどうぞよろしくお願いいたします！✨
