# 虎の巻：目的別デジタルツール使い分けマップ

「今、これをやりたい！」と思った時にの、最適な道具（ツール）が一目でわかる早見表です。
文子さんの「もっている武器」を整理しました。

---

## 🏢 1. 仕事・コミュニケーション編

| やりたいこと | 使うツール | イメージ |
| :--- | :--- | :--- |
| **お客さんの情報を知りたい**<br>（電話する前、履歴確認） | **Salesforce**<br>（セールスフォース） | 📒 **最強の顧客名簿**<br>全員で共有する営業日報。 |
| **チーム・部活で連絡したい**<br>（すぐ返事ほしい、雑談） | **Slack**<br>（スラック） | 👔 **仕事用チャット**<br>スーツを着たLINE。スタンプで会話。 |
| **趣味・息子さんと話したい**<br>（ゲーム、雑談、通話） | **Discord**<br>（ディスコード） | 🎮 **遊び場チャット**<br>パーカーを着たLINE。電話しっぱなしOK。 |
| **文書・表・メールを作りたい**<br>（見積書、マニュアル、連絡） | **Google Workspace**<br>（ワークスペース） | 🏗️ **デジタルオフィス**<br>どこでも使えるWord/Excel/メール。 |

---

## 🤖 2. 生成AI・クリエイティブ編

| やりたいこと | 使うAIツール | イメージ |
| :--- | :--- | :--- |
| **最新情報を調べたい**<br>（検索、リサーチ、比較） | **Genspark**<br>（ジェンスパーク） | 🔍 **超・検索エンジン**<br>複数のサイトをまとめて要約してくれる。 |
| **何かをゼロから知りたい**<br>（質問、アイデア出し） | **Perplexity**<br>（パープレキシティ） | 🧠 **物知り博士**<br>対話しながら深掘りして教えてくれる。 |
| **資料・スライドを作りたい**<br>（プレゼン、企画書） | **Gamma**<br>（ガンマ） | 🎨 **資料デザイナー**<br>文字を入れるだけでおしゃれなスライドが完成。 |
| **図解・ポンチ絵を描きたい**<br>（手書きメモの清書） | **Napkin**<br>（ナプキン） | ✏️ **図解職人**<br>テキストを図やチャートにしてくれる。 |
| **おしゃれな画像が欲しい**<br>（チラシ、LPの挿絵） | **Canva**<br>（キャンバ） | 🖼️ **デザイン画材屋**<br>テンプレートを選んで写真や文字を配置。 |
| **コード・LPを書きたい**<br>（サイト制作、アプリ開発） | **Antigravity**<br>（私です！） | 🦸 **開発パートナー**<br>「第二の脳」として一緒にモノ作り。 |

---

## 🚦 使い分けのコツ（迷ったら）

1.  **「人」**と関わるなら？
    *   社内の人 → **Slack**
    *   お客さん → **Salesforce** か **Gmail**

2.  **「モノ」**を作りたいなら？
    *   とりあえず調べる → **Genspark**
    *   形にする（資料） → **Gamma**

3.  **「自分」**の思考を整理するなら？
    *   日誌やメモ → **Obsidian（私のわたし）**
