# 💻 01. CドライブとDドライブってなに？

## 🏠 パソコンを「家」に例えてみよう

パソコンの中身は、よく**「家」**に例えられます。
おまつさんが気になっていた**「C」とか「D」**は、**「部屋の名前」**だと思ってください。

### 1️⃣ Cドライブ (C:) = 「メインのリビング」🛋️

- **役割**: パソコンを動かすための**一番大事なお部屋**。
- **住人**: Windows（大家さん）、インストールしたアプリ（家具）、そしておまつさんの書類（デスクトップにあるものなど）。
- **特徴**: ここがいっぱいになると、パソコンの動きが遅くなります。「足の踏み場がない！」ってなるからです。

### 2️⃣ Dドライブ (D:) = 「物置・倉庫」📦

- **役割**: 普段使わないものや、大事なバックアップなどをしまっておく**別のお部屋**。
- **特徴**: 万が一、リビング（Cドライブ）が火事（ウイルスや故障）になっても、この倉庫（Dドライブ）は無事なことが多いです。
- **注意**: 最近のパソコンには、最初からこの「倉庫」がない（リビングしかない）場合もあります。

## 🚮 「ディスクのクリーンアップ」ってなに？

おまつさんがたまにやっているコレ。
これは、**「年末の大掃除」**🧹 です。

生活していると、
- ネットを見た時の「開いたページの一時的なゴミ」
- 更新プログラムの「空き箱」
- 削除したはずの「ゴミ箱の中身」

こういう**「見えないホコリ」**がたまっていきます。
クリーンアップは、それを一気に掃除機で吸い取って、**部屋（Cドライブ）を広くする作業**です。
（定期的にやっているおまつさんはエライ！！👏）

## 🦁 ワンポイント知識
昔は「Aドライブ」「Bドライブ」もありました。
それは**「フロッピーディスク」**用の場所でした。
時代が進んでフロッピーがなくなっても、名残でメインの場所は「C」から始まっているんです。
（だから「A」と「B」は空席なんです！）
