# Googleカレンダー連携ガイド

「第2の脳（ローカル）」の予定を、スマホなどのGoogleカレンダーに送る方法です。

## 1. 予定を書き出す
チャットで以下のコマンドを入力してください。
> `/export-schedule`
（または「カレンダー用に書き出して」と話しかけてください）

AIが現在のスケジュールを読み取り、連携用ファイルを作成します。

## 2. ファイルを取り込む (Googleカレンダーの場合)
ダブルクリックで「Outlook」などの別アプリが開いてしまう場合は、以下の手順（ブラウザからのインポート）が確実です。

1.  パソコンで **[Googleカレンダー](https://calendar.google.com/)** を開く。
2.  右上の歯車アイコン ⚙️ をクリックし、**「設定」** を選ぶ。
3.  左メニューの **「インポート / エクスポート」** をクリック。
4.  「パソコンからファイルを選択」をクリックし、`06_Schedule/Export/schedule_export.ics` を選ぶ。
5.  **「インポート」** ボタンを押す。

これで確実にGoogleカレンダーに追加されます！
