# スケジュール写真取り込みガイド

紙のスケジュール帳やカレンダーを「第2の脳」に取り込むための手順です。

## 1. 写真を撮る
スマホで手帳のページやカレンダーを撮影します。
*   **ポイント**: 文字が読めるように、明るい場所で、できるだけ正面から撮影してください。

## 2. 写真を保存する
撮影した写真を、以下のフォルダに保存してください。
`06_Schedule/Inbox`

※スマホからPCへの転送方法が不明な場合は、OneDriveアプリを入れるか、自分宛てのメール/LINEで送るなどが簡単です。

## 3. 合図する
写真を入れたら、私（AI）にチャットでこう伝えてください。
> 「スケジュールの写真を入れたよ。読み取って！」

## 4. 自動処理
私が以下の処理を行います。
1.  画像の文字を読み取ります（OCR）。
2.  内容を解析し、デジタルテキスト化します。
3.  `2026_Yearly_Plan.md` などの適切なファイルに予定を追記します。
4.  （必要に応じて）Googleカレンダー用のファイルを生成します。

---
**準備完了！まずは写真を入れてみてください。**
