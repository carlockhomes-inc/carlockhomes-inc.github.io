# 17_AI導入・活用リスクアセスメントシート

## 📌 本シートの目的
新たなAI技術を導入またはシステム開発に活用する際、MIT Sloan（CAMS）基準のリスクを多角的に検証し、安全な「アクセル全開」を可能にするためのチェックリストです。

---

### 1. 技術的リスク（Technical Risk）
*   [ ] **セキュリティ**: データの入力・漏洩プロトコルは確立されているか？
*   [ ] **バイアス（偏り）**: 特定の属性（年齢、地域等）に対する不当な判断が混入していないか？
*   [ ] **ハルシネーション（虚偽）**: AIの嘘を検知し、人間が最終確認する仕組みがあるか？

### 2. 社会的リスク（Social Risk）
*   [ ] **プライバシー**: 社員や顧客の機微情報が、AIの学習や外部へ流出する設計になっていないか？
*   [ ] **不正利用の防止**: ツールが意図しない目的（誹謗中傷、情報抜き取り等）に使われる隙はないか？
*   [ ] **社会的受容性**: 既存の社員の心理的な不安や、世間的な倫理観に抵触していないか？

### 3. 法的リスク（Legal Risk）
*   [ ] **知的財産（IP）**: 学習データや生成物の著作権侵害リスクをクリアしているか？
*   [ ] **説明責任（AIMS）**: 万が一のトラブル時、AIの判断プロセスを法的に説明できるか？
*   [ ] **安全・品質担保**: AIの判断に起因する事故の際の、責任所在は明確か？

---

### 💡 判定ガイド
*   **ALL-CHECK**: アクセルを全開に踏み込んでください。
*   **1-2項目の未チェック**: 導入を保留し、該当リスクの回避策（サスペンション）を強化してください。
*   **3項目以上の未チェック**: 重大な経営リスクあり。戦略を根本から見直してください。
