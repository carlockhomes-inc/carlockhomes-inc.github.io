# Manual Architect (マニュアル建築家)

## 概要
このスキルは、fumikoさんが作成した「業務手順のメモ」や「Markdown下書き」を、そのまま印刷したり共有できる「プロフェッショナルなHTMLマニュアル」に変換します。
タグ打ちやCSSデザインは一切不要です。

## 使い方
1.  **マニュアルにしたいファイルを開く**:
    *   変換したいMarkdownファイル（`.md`）またはテキストを開いた状態にします。
2.  **スキルを呼び出す**:
    *   チャットで「このファイルをマニュアルにして」「HTMLに変換して」と指示します。
3.  **オプション指定（任意）**:
    *   「赤と青の色分けにして（重要/通常）」
    *   「フローチャートを入れて」
    *   「印刷用にA4で区切って」
    など、好みのデザインを伝えてください。

## 出力物
*   同一フォルダに `.html` ファイルとして保存されます。
*   自動的にブラウザでプレビューされます。

## デザインの特徴
*   **Visual First**: 手順をただ並べるだけでなく、視覚的にわかりやすい「ボックスデザイン」や「アイコン」を自動付与します。
*   **Print Ready**: A4用紙で印刷した際に崩れない印刷用CSS（`@media print`）が組み込まれています。
*   **Responsive**: スマホで見ても崩れません。

## Technical Details (AI Instructions)
このスキルを実行するAIは、以下の手順に従ってください。

1.  **Read Content**: ターゲットファイルのMarkdown内容を解析する。
2.  **Transpile to HTML**:
    *   基本構造: HTML5 + 内包CSS (Single File HTML)。
    *   CSSフレームワーク: なし（Vanilla CSS推奨）。fumiko好みの「柔らかいがスッキリしたデザイン」を採用。
    *   配色: 
        *   メイン: 濃紺 (#2C3E50) - 信頼感
        *   アクセント: サーモンピンク (#FF6B6B) - 警告/注意
        *   背景: オフホワイト (#F9F9F9) - スッキリ感
3.  **Enhance Visualization**:
    *   `mermaid` ブロックがある場合は、`mermaid.js` CDNリンクを含めてレンダリング可能にする。
    *   手順（List）は、数字付きのカードデザイン（Step UI）に変換する。
4.  **Save & Report**:
    *   変換後のファイルを保存し、パスを提示する。
