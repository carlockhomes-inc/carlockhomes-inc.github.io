# プロジェクト引継ぎ書：Carlock Homes 公式サイト・リニューアル

このドキュメントは、公式サイトのモダナイズとSNS連携の全工程を記録したものです。新しいPC環境や、今後の開発・運用時に参照してください。

---

## 🏆 プロジェクトの概要
- **クライアント**: Carlock Homes (CLH)
- **主要目的**: 
    - 既存の静的サイトを、信頼感と高級感のあるハイエンドなデザインへ刷新。
    - Facebook / Instagram との強力な連携（QRコード活用）。
    - チーム運営を可能にする「運用マニュアル」の整備。

## 📍 重要なファイル構造
- **[index.html](index.html)**: 
    - 本番のトップページ。
    - **機能**: シネマティックヒーロー、事業紹介カード、最新ニュース、スクロール追従UI、大型QRコード付きSNSセクション。
- **[pages/company.html](pages/company.html)**: 
    - 会社案内（スライド形式）。
- **[demo/](demo/)**: 
    - クリエイティブチーム向けのデモ素材（スマホ版・PC版の映像とキャプチャ）。
- **[MANUAL.html](MANUAL.html)**: 
    - チーム向けの運用マニュアル。ブラウザで閲覧可能。

## 🎨 デザインのこだわり
- **Branding**: 綴りを `Carlock Homes` に統一。略称は `CLH`。
- **Theme**: ネイビー（紺）とゴールド（金）を基調とした、プロフェッショナルかつラグジュアリーな配色。
- **SNS Strategy**: 
    - PC版：スマホで撮影しやすいよう、QRコードを大きく（200px）表示。
    - スマホ版：QRコードを自動非表示にし、直接URLを叩ける直感的なボタンUIへ切り替え。

## 🛠️ 技術スタック
- HTML5 / CSS3 (Vanilla CSS) / JavaScript (Vanilla JS)
- 外部ライブラリ：Google Fonts (Outfit, Crimson Text, Zen Kaku Gothic New)
- お問い合わせ：Googleフォームへの外部リンク。
