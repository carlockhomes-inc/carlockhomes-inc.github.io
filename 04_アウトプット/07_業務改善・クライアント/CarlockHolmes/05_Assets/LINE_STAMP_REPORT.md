## 🖼 スタンプ・ラインナップ（全16種）

### 1. セキュリティ・業務報告（カメラちゃん中心）
*   **「見守っています」**: 防犯カメラとして優しく見守る表情
*   **「異常なし！」**: キリッと敬礼するカメラちゃん
*   **「確認しました」**: レンズを覗き込んで確認
*   **「完了しました！」**: モニタリングや作業が完了した合図

### 2. お客様・現場連絡用
*   **「今向かってます！」**: 現場に急行するイメージ
*   **「鍵かけた？」**: カギ君による防犯啓蒙（代表作）
*   **「鍵忘れないで」**: お出かけ前のリマインド

### 3. 日常挨拶・感情
*   **おはよう / お疲れ様**: 日常のやり取りをキャラクターで
*   **ありがとう / よろしく**: 感謝とお願い
*   **OK / 了解**: 実用的な返信ツール
*   **LOVE / SHOCK / スヤァ**: 愛着を持ってもらうための表情

## 🚀 次のステップ：LINEクリエイターズマーケットへの登録

LINEスタンプとして公開するためには、以下のサイズ調整と余白設定が必要です。

### 📋 LINEスタンプの公式規格
| 項目 | 指定サイズ / 詳細 |
| :--- | :--- |
| **スタンプ画像** | **最大 370 × 320 px** （個数：8/16/24/32/40個） |
| **メイン画像** | **240 × 240 px** （1個・販売ページ用） |
| **トークルームタブ画像** | **96 × 74 px** （1個・選択タブ用） |
| **余白** | 上下左右に **10px程度** の余白が必要 |
| **形式** | 背景透過PNG |
