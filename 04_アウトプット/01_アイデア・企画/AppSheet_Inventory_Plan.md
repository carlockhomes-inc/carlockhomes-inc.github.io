# 【AppSheet用】在庫管理アプリ・構成案

# 1. 概要
# このファイルは、Googleスプレッドシートにコピペして使用するための設計図です。
# 以下の2つのシート（タブ）を作成し、1行目に項目名を入力してください。

# ---------------------------------------------------------
# シート1: 部材マスタ (Items)
# ---------------------------------------------------------
# 説明: 在庫管理したい部材のリストです。
#
# A列: 商品コード (Key)
#      - 例: A-001, B-105
#      - ※これがスマホで読み取る対象です。重複しないようにしてください。
#
# B列: 商品名
#      - 例: ネジ(大), 梱包テープ
#
# C列: カテゴリ
#      - 例: 金具, 消耗品
#
# D列: 商品画像
#      - ※スマホで写真を撮って登録できます（画像型の列として設定）。
#
# E列: 現在在庫数 (Virtual Columnで計算推奨だが、初期値として用意)
#      - 例: 100
#
# F列: 最低在庫数 (これ以下になったらアラート的な使い方も可能)
#      - 例: 10

# ---------------------------------------------------------
# シート2: 入出庫ログ (Logs)
# ---------------------------------------------------------
# 説明: 「いつ」「誰が」「何を」「どうしたか」の記録です。
#
# A列: ログID (Key)
#      - ※AppSheetで「UNIQUEID()」を設定し、自動採番させます。
#
# B列: 日時
#      - ※AppSheetで「NOW()」を設定し、自動入力させます。
#
# C列: 商品コード (Ref)
#      - ※部材マスタのA列を参照します。
#      - ※ここでスマホのキーボードを使って文字スキャンします。
#
# D列: 区分
#      - ※「入庫」「出庫」「棚卸」から選択ボタンにします。
#
# E列: 数量
#      - ※入庫ならプラス、出庫ならマイナスの数を入力します。
#        (またはアプリ側で計算ロジックを入れる)
#
# F列: 担当者
#      - ※「USEREMAIL()」で自動入力、またはリストから選択。
#
# G列: 備考
#      - ※メモ欄。

# ---------------------------------------------------------
# 💡 これからの手順
# ---------------------------------------------------------
# 1. 上記の内容でGoogleスプレッドシートを作成する。
# 2. スプレッドシートのメニュー「拡張機能」>「AppSheet」>「アプリを作成」を押す。
# 3. AppSheetの画面が開いたら、以下の設定をする（AIがサポートします）。
#    - 部材マスタと入出庫ログのテーブル追加
#    - Column設定（型定義）
#    - Virtual Columnでの在庫計算式の設定
