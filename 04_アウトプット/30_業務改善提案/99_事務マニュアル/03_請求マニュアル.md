# 💸 請求マニュアル

## 1. 請求業務の概要
毎月の売上を確定し、正確に請求書を発行・送付するための手順です。

---

## 2. 月次スケジュール

| 日程 | 業務内容 | 担当者 |
| :--- | :--- | :--- |
| **毎月 1日** | 前月分の作業報告書の回収・チェック | 事務担当 |
| **毎月 5日** | 請求金額の確定・システム入力 | 経理担当 |
| **毎月10日** | 請求書発行・送付 (PDF/郵送) | 事務担当 |
| **毎月20日** | [ToDo] | |

---

## 3. クライアント別 締め日・必着日リスト

| クライアント名 | 締め日 | 請求書必着日 | 送付方法 | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| **マルイ様** | 月末 | 翌月3営業日 | PDFメール | 担当：田中様宛 |
| **B社** | 20日 | 25日 | 郵送 | 原本が必要 |
| **C社** | 月末 | 翌月5日 | システム | 専用WEBサイトに入力 |

---

## 4. 請求書作成時のチェックポイント
*   [ ] 工事番号・件名は合っているか？
*   [ ] 金額（税抜・税込）に相違はないか？
*   [ ] 宛名は正しいか？（部署名変更などに注意）
*   [ ] 振込口座の記載はあるか？

---

## 5. 入金確認
*   毎月末に、前月請求分の入金を確認し、消込を行う。
