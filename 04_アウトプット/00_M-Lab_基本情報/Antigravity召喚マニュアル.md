# 【極秘手順書】新しいPCへの『Antigravity（No.3）』召喚マニュアル

このマニュアルは、新しいPC（または別のPC）に、Fumiko専属執事である「私（Antigravity）」を召喚するための手順書です。

---

## 🕒 召喚のための3ステップ

### 1. 新しいPCでエディタを開く
まずは、新しいPCで以下のエディタ（開発ツール）を開きます。
*   **VS Code (Visual Studio Code)**
*   または **Project IDX**
*   または **Google Cloud Code**

※今使っている「古いPC」と同じ環境を作るなら、**「Import VS Code Settings（設定をインポート）」** を選ぶのが最短ルートです！

### 2. 拡張機能（Extension）から私を探す
画面の左端にあるバーを見てください。
アイコンが縦に並んでいる中に、**「田」のような四角いマーク（拡張機能ボタン）** があります。

1.  **「田」マーク** をクリック。
2.  上に出てくる**検索窓**に、以下の呪文（名前）を入力します。
    *   **`Antigravity`**
    *   もし見つからない場合は：**`Project IDX`** または **`Google Cloud Code`**

**⚠️ 偽物に注意！**
*   アイコンが「今の私（六角形やAの字のような幾何学模様）」と同じか確認！
*   作者名（Publisher）が **`Google`** であることを確認！

### 3. インストール（召喚！）
本物の私を見つけたら、**「Install（インストール）」** ボタンをポチッ！

インストールが完了すると、画面に私のアイコンが現れます。
そこをクリックして、
**「Sign in with Google（Googleでサインイン）」**
を選び、いつものFumikoさんのアカウントでログインしてください。

---

## ⚙️ 初期設定（召喚後の儀式）

ログイン後、私からいくつか質問されますが、以下の通りに答えてください。
（Fumikoさんにとって一番使いやすい「標準設定」です）

1.  **How do you want to use...?（どうやって使う？）**
    *   👉 **`Review-driven development (RECOMMENDED)`** を選択
    *   （「私が勝手に動く前に、Fumikoさんに確認するよ」という設定）

2.  **Configure Your Editor（エディタの設定）**
    *   👉 何も変えずに **`Next`** を押す
    *   （Keybindings: Normal, Extensions: Recommended のままでOK！）

3.  **Trust Authors（作者を信頼する？）**
    *   フォルダを開いた時に聞かれたら、
    *   👉 **`Yes, I trust the authors`** を選択
    *   （「この2nd-Brainフォルダは安全だよ！」という合図）

これで召喚完了です！
新しいPCの私も、古いPCの私と同じ記憶（OneDrive）を持って、いつでもFumikoさんをお待ちしています！🎩✨
